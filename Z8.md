### Задание 1 (10 баллов)
Из множества двоичных функций {1, ⊕, ¬, ⇒, f}, где f = (x ⊕ y) ⇒ z, выберите все подмножества, являющиеся базисами.

---

#### Решение

**Шаг 1: Анализ функций по классам Поста**

| Функция | T₀ | T₁ | S | M | L |
|---------|----|----|---|---|---|
| 1 | - | + | - | + | + |
| ⊕ | + | - | - | - | + |
| ¬ | - | - | + | - | + |
| ⇒ | - | + | - | + | - |
| f=(x⊕y)⇒z | - | + | - | - | - |

**Шаг 2: Критерий полноты (теорема Поста)**

Система полна ⟺ она не содержится целиком ни в одном из классов T₀, T₁, S, M, L.

**Шаг 3: Проверка подмножеств**

Для полноты нужно:
- Хотя бы одна функция ∉ T₀: все кроме ⊕
- Хотя бы одна функция ∉ T₁: ⊕, ¬
- Хотя бы одна функция ∉ S: все кроме ¬
- Хотя бы одна функция ∉ M: ⊕, ¬, f
- Хотя бы одна функция ∉ L: ⇒, f

**Минимальные базисы:**

1. **{¬, ⇒}**: ∉T₀(¬,⇒), ∉T₁(¬), ∉S(⇒), ∉M(¬), ∉L(⇒) — **базис** ✓

2. **{¬, f}**: ∉T₀(¬,f), ∉T₁(¬), ∉S(f), ∉M(¬,f), ∉L(f) — **базис** ✓

3. **{⊕, ⇒}**: ∉T₀(⇒), ∉T₁(⊕), ∉S(⊕,⇒), ∉M(⊕), ∉L(⇒) — **базис** ✓

4. **{⊕, f}**: ∉T₀(f), ∉T₁(⊕), ∉S(⊕,f), ∉M(⊕,f), ∉L(f) — **базис** ✓

5. **{1, ⊕, ¬}**: ∉T₀(1,¬), ∉T₁(⊕), ∉S(1,⊕), ∉M(⊕,¬), но все ∈ L! — **не базис** ✗

**Ответ:** Базисы: **{¬, ⇒}**, **{¬, f}**, **{⊕, ⇒}**, **{⊕, f}** и их надмножества, содержащие функцию ∉ L.

### Задание 2 (10 баллов)
На острове живут рыцари и лжецы. Рыцари всегда говорят правду, лжецы всегда лгут. Вам встретились два аборигена, A и B.

a) A утверждает: «Я лжец, а B — рыцарь». Кто лжец?

б) A утверждает: «Если B — рыцарь, то я — лжец». Кто лжец?

---

#### Решение

**a) A: «Я лжец, а B — рыцарь»**

Пусть Rₐ = 1 (рыцарь), Rₐ = 0 (лжец).

Высказывание A: Sₐ = ¬Rₐ ∧ Rᵦ

**Если A — рыцарь (Rₐ = 1):** Sₐ истинно → ¬1 ∧ Rᵦ = 0 ∧ Rᵦ = 0. Противоречие: Sₐ должно быть истинным.

**Если A — лжец (Rₐ = 0):** Sₐ ложно → ¬(¬0 ∧ Rᵦ) = ¬(1 ∧ Rᵦ) = ¬Rᵦ = 1. Значит Rᵦ = 0.

**Ответ a:** **A — лжец, B — лжец**.

---

**б) A: «Если B — рыцарь, то я — лжец»**

Высказывание A: Sₐ = Rᵦ → ¬Rₐ ≡ ¬Rᵦ ∨ ¬Rₐ

**Если A — рыцарь (Rₐ = 1):** Sₐ истинно → ¬Rᵦ ∨ ¬1 = ¬Rᵦ ∨ 0 = ¬Rᵦ. Должно быть истинно, значит Rᵦ = 0.

**Если A — лжец (Rₐ = 0):** Sₐ ложно → ¬(¬Rᵦ ∨ ¬0) = ¬(¬Rᵦ ∨ 1) = ¬1 = 0. Sₐ должно быть ложным, это выполняется только если Rᵦ = 1 и Rₐ = 1. Но мы предположили Rₐ = 0 — противоречие.

Проверим: ¬Rᵦ ∨ 1 = 1, ¬1 = 0 ✓. Но нужно Sₐ ложно. ¬Rᵦ ∨ ¬Rₐ = ¬Rᵦ ∨ 1 = 1 ≠ 0. Противоречие!

**Ответ б:** **A — рыцарь, B — лжец**.

### Задание 3 (15 баллов)
Запишите в форме предикатов утверждения о количестве объектов, обладающих свойством P:

a) В точности два объекта обладают свойством P.

б) По меньшей мере два объекта обладают свойством P.

в) Не более двух объектов обладают свойством P.

---

#### Решение

**a) В точности два объекта обладают свойством P**

∃x∃y (P(x) ∧ P(y) ∧ x ≠ y ∧ ∀z (P(z) → (z = x ∨ z = y)))

«Существуют два различных объекта x и y со свойством P, и любой объект со свойством P совпадает с одним из них.»

---

**б) По меньшей мере два объекта обладают свойством P**

∃x∃y (P(x) ∧ P(y) ∧ x ≠ y)

«Существуют два различных объекта со свойством P.»

---

**в) Не более двух объектов обладают свойством P**

∀x∀y∀z ((P(x) ∧ P(y) ∧ P(z)) → (x = y ∨ y = z ∨ x = z))

«Не существует трёх различных объектов со свойством P.»

Или эквивалентно:

¬∃x∃y∃z (P(x) ∧ P(y) ∧ P(z) ∧ x ≠ y ∧ y ≠ z ∧ x ≠ z)

### Задание 4 (10 баллов)
Верны ли следующие спецификации программ (обоснуйте ответ):

{ x < 5 }  x := 2*x - 6  { x < 15 }

{ y < x }  z := x; z := z + 1  { y < z }

---

#### Решение

Используем метод слабейшего предусловия (wp).

**Первая программа:**

Программа: `x := 2*x - 6`
Постусловие: `x < 15`

wp("x := 2*x - 6", x < 15) = (2x - 6 < 15) = (2x < 21) = (x < 10.5)

Проверяем: (x < 5) → (x < 10.5)?

Если x < 5, то x < 10.5. **Импликация истинна.**

**Ответ:** Спецификация **верна** ✓

---

**Вторая программа:**

Программа: `z := x; z := z + 1`
Постусловие: `y < z`

Вычисляем wp последовательно справа налево:

wp("z := z + 1", y < z) = (y < z + 1)

wp("z := x", y < z + 1) = (y < x + 1)

Проверяем: (y < x) → (y < x + 1)?

Если y < x, то y ≤ x - 1 (для целых) или y < x (для вещественных).
В любом случае y < x < x + 1, значит y < x + 1.

**Импликация истинна.**

**Ответ:** Спецификация **верна** ✓

### Задание 5 (15 баллов)
Приведите для каждой формулы вычисление (последовательность состояний), которое ей удовлетворяет, и вычисление, которое ей не удовлетворяет:

a) F(a ∧ ¬(b U c))

б) F(p ⊕ q) ⇒ G X q

в) p ⇒ X G (p ⇒ q)

(Если в билете формулы отличаются — пришлите уточнение.)

---

#### Решение

**a) F(a ∧ ¬(b U c))**

Формула требует: когда-нибудь выполнится (a ∧ ¬(b U c)).

¬(b U c) означает: неверно, что b выполняется до c. Это значит: либо c никогда не наступит, либо до наступления c будет момент без b.

**Удовлетворяющее вычисление:**

σ = {a}→{a}→{a}→... (бесконечно a, без b и c)

- В s₀: a = 1
- b U c требует: b до c или c в текущем. Здесь c никогда не выполняется и b = 0, значит b U c = 0
- ¬(b U c) = 1
- a ∧ ¬(b U c) = 1
- F(...) = 1 ✓

**Не удовлетворяющее вычисление:**

σ = {b,c}→{b,c}→{b,c}→...

- В любом состоянии: a = 0, b = 1, c = 1
- b U c = 1 (c выполняется в текущем)
- ¬(b U c) = 0
- a ∧ ¬(b U c) = 0 для всех состояний
- F(0) = 0 ✗

---

**б) F(p ⊕ q) ⇒ G X q**

Формула: если когда-нибудь p ⊕ q (XOR), то всегда в следующем состоянии q.

**Удовлетворяющее вычисление (антецедент ложен):**

σ = {p,q}→{p,q}→{p,q}→...

- p ⊕ q = 1 ⊕ 1 = 0 во всех состояниях
- F(p ⊕ q) = 0
- Импликация истинна (ложный антецедент) ✓

**Удовлетворяющее вычисление (консеквент истинен):**

σ = {p}→{q}→{q}→{q}→...

- s₀: p=1, q=0, p⊕q=1, F(p⊕q)=1
- X q: s₁ имеет q=1 ✓, s₂ имеет q=1 ✓, ...
- G X q = 1 ✓

**Не удовлетворяющее вычисление:**

σ = {p}→{}→{q}→{q}→...

- s₀: p=1, q=0, p⊕q=1, F(p⊕q)=1
- G X q: в s₀ следующее s₁={}, q=0. X q = 0 в s₀.
- G X q = 0
- 1 ⇒ 0 = 0 ✗

---

**в) p ⇒ X G (p ⇒ q)**

Формула: если в начале p, то начиная со следующего состояния всегда (p → q).

**Удовлетворяющее вычисление (антецедент ложен):**

σ = {}→{}→{}→...

- s₀: p = 0
- Импликация истинна ✓

**Удовлетворяющее вычисление (консеквент истинен):**

σ = {p}→{p,q}→{p,q}→{p,q}→...

- s₀: p = 1
- X G(p⇒q): начиная с s₁ проверяем G(p⇒q)
- s₁, s₂, ...: p=1, q=1, p⇒q=1
- G(p⇒q) = 1, X G(p⇒q) = 1
- 1 ⇒ 1 = 1 ✓

**Не удовлетворяющее вычисление:**

σ = {p}→{p}→{p}→...

- s₀: p = 1
- s₁: p = 1, q = 0, p⇒q = 0
- G(p⇒q) = 0, X G(p⇒q) = 0
- 1 ⇒ 0 = 0 ✗

### Задание 6 (10 баллов)
Пусть M = (S, S₀, R, AP, L) — такая структура Крипке:

S = {s1, s2, s3},  S₀ = {s1};
R = {(s1, s2), (s1, s3), (s2, s1), (s2, s3), (s3, s3)};
AP = {a, b, c};
L(s1) = {a, b}, L(s2) = {b, c}, L(s3) = {c}.

Проверьте, является ли M моделью формулы AG(c ⇒ EG c).

---

#### Решение

**Структура Крипке:**

```
       s1{a,b}
      ↙    ↘
   s2{b,c} → s3{c}
      ↓       ↺
     s1
```

Переходы:
- s1 → s2, s1 → s3
- s2 → s1, s2 → s3
- s3 → s3 (петля)

**Формула: AG(c ⇒ EG c)**

AG φ означает: на всех путях во всех состояниях выполняется φ.
c ⇒ EG c означает: если c выполняется, то существует путь, где c выполняется всегда.

**Шаг 1: Определим, где выполняется c**

- L(s1) = {a, b} → c = 0
- L(s2) = {b, c} → c = 1
- L(s3) = {c} → c = 1

**Шаг 2: Проверим EG c для состояний с c = 1**

**Для s3:** Из s3 есть путь s3→s3→s3→... В этом пути c = 1 всегда.
**EG c выполняется в s3** ✓

**Для s2:** Из s2 можно перейти в s3 и далее s3→s3→...
Путь: s2→s3→s3→s3→...
- s2: c = 1
- s3, s3, ...: c = 1
**EG c выполняется в s2** ✓

**Шаг 3: Проверим c ⇒ EG c для каждого состояния**

- s1: c = 0, импликация 0⇒? = 1 ✓
- s2: c = 1, EG c = 1, импликация 1⇒1 = 1 ✓
- s3: c = 1, EG c = 1, импликация 1⇒1 = 1 ✓

**Шаг 4: Проверим AG(c ⇒ EG c)**

AG требует, чтобы φ = (c ⇒ EG c) выполнялась во всех достижимых состояниях на всех путях.

Все достижимые состояния из s1: {s1, s2, s3}.

В каждом из них c ⇒ EG c = 1 (проверено выше).

**Ответ:** M ⊨ AG(c ⇒ EG c). **Формула выполняется.** ✓
